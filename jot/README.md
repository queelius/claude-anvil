# jot — Claude Code Plugin

Journal-aware Claude Code sessions. Surfaces relevant tasks, ideas, and plans
from your jot journal on session start, providing ambient context awareness
across conversations.

## Components

| Type | Name | Purpose |
|------|------|---------|
| Skill | `jot` | Journal query patterns and context surfacing |

## Install

```bash
# From the marketplace
/plugin install jot@queelius
```

## How It Works

Jot integrates your personal journal with Claude Code sessions. On session
start, it surfaces relevant entries — open tasks, recent ideas, active plans —
so Claude has context about what you're working on without you having to
re-explain.
